## 🛠 Technologies utilisées

Ce projet a été développé avec **Vue 3**, **Vite** et **TypeScript**.

- **Framework :** [Vue.js 3](https://vuejs.org/) (SFCs avec `<script setup>`)
- **Routage :** Vue Router
- **Build Tool :** [Vite](https://vitejs.dev/)
- **Langage :** TypeScript

*Note : Mon socle technique global s'étend particulièrement sur le Back-end avec Java/SpringBoot, Node.js et PostgreSQL.*
